# Купон — New user

Купон только для новых клиентов — на **первую** аренду. Инструмент привлечения.

| Параметр | Значение |
|---|---|
| Кому | Клиент без прошлых `charged`-ордеров |
| QR | Опционально |
| Сегментация | По странам |
| Эффект | Обычно крупная скидка (`percent`, вплоть до −100%) или `balance_credit` |

## Особая проверка
Помимо стандартного пайплайна — **у клиента не должно быть ни одной оплаченной аренды** (`orders`, статус `charged`). Иначе отказ.

> [!warning] Антифрод
> Главный риск — мульти-аккаунты ради повторного «первого» бонуса. Нужна OTP-верификация телефона и контроль совпадений устройства/карты ([[Открытые вопросы]]). Пересекается с [[Приветственный бонус]] — выбрать один механизм для новичка.

## Пример
«Первая аренда бесплатно»: `percent=100` → аренда 1.80 € к оплате 0.00. Сплит на 0.00.
