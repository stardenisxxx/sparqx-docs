# Технический стек

| Категория | Технология | Детали |
|---|---|---|
| Frontend Веб | React + Vite + TypeScript + Tailwind | Vite 6.4, React 18 |
| Frontend Mobile | React Native + Expo | Expo SDK 51 |
| Графики | Recharts | Area, Donut, Bar, Heatmap |
| Drag & Drop | @hello-pangea/dnd | Kanban + сеты рекламы |
| QR | qrcode.react | Генерация QR купонов |
| Icons | lucide-react | Единый набор |
| Backend | Node.js / Go / Java | Зависит от сервиса |
| Database | PostgreSQL 16 | 6 схем, репликация, партиции |
| Cache | Redis 7 | Кластер master + 2 реплики |
| Queue | RabbitMQ 3.12 | 6 очередей `durable:true` |
| Storage | MinIO | S3-совместимое |
| Orchestration | Kubernetes + Docker | 1 control + 2 worker |
| Load Balancer | HAProxy | Active + Standby |
| IoT Broker | Alibaba Cloud IoT (MQTT) | ProductKey, порт 1883 |
| Monitoring | Prometheus + Grafana + ELK | 3 дашборда + Alertmanager |
| CI/CD | GitLab CI | Rolling update через kubectl |
