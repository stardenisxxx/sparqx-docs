# Микросервисы

10 сервисов в Kubernetes. Каждый отвечает за свою зону, общаются через API Gateway и очереди RabbitMQ.

| Сервис | Инстансов | Зона ответственности |
|---|---|---|
| api-gateway | 3 | JWT авторизация, маршрутизация, rate limiting |
| auth-service | 2 | Генерация JWT, сессии в Redis, RBAC |
| device-service | 5 | MQTT, телеметрия `0x10`, команды на устройства |
| rental-service | 3 | Создание ордеров, выдача, возврат |
| finance-service | 3 | Расчёт стоимости, [[Франшизы и сплит\|сплит]], payouts |
| user-service | 2 | Клиенты, администраторы, роли |
| coupon-service | 2 | Валидация и применение [[Купоны\|купонов]] |
| media-service | 2 | [[Реклама\|Рекламные кампании]], доставка на устройства |
| support-service | 2 | [[Поддержка\|Тикеты]], переписка, вложения |
| notification-service | 2 | Push-уведомления, алерты |

## Очереди RabbitMQ
`telemetry` · `payments` · `splits` · `notifications` · `media` · `alerts` — все `durable:true`.
