# Открытые вопросы

Бизнес-решения, которые ещё не зафиксированы. Поля в БД заложены, значения/правила — нет.

| Вопрос | Контекст | Статус |
|---|---|---|
| Межстрановой возврат | Берут в стране A, возвращают в B — кому выручка? | Поле `cross_country` в payments, правило не зафиксировано → [[Франшизы и сплит]] |
| Минимальный порог выплаты | Ниже какой суммы не формировать payout? | Поле в Schedule payment, значение не задано → [[Финансы]] |
| Автоперевод денег | Payouts через банк API или только отметка в UI? | Пока ручная отметка, банк API — следующий этап |
| Значения TTL по видам бонусов | Единый срок или разный для referral/coupon/promo? | TTL есть (решено), конкретные значения — нет → [[Бонусы]] |
| Потолок «% оплаты бонусом» | Можно ли оплатить 100% бонусом или есть лимит на транзакцию? | Не зафиксировано → [[Бонусы]] |
| Welcome / кешбэк / лояльность | Вводим ли эти бонусы, какие %/пороги? | Идеи на перспективу → [[Бонусы]] |

## Решённые
| Вопрос | Решение |
|---|---|
| Роль баланса и вывод | Гибрид real/bonus; выводимы только внесённые деньги → [[Баланс]] |
| Подписка = % или часы | **Скидка %** на весь тариф (`discount_pct`) → [[Подписки и тарифы]] |
| Реферальная программа | Бонус получают **оба** участника, в bonus-карман → [[Купоны]], [[Баланс]] |
| Срок жизни бонусов и лимиты вывода | **Да**: бонусы сгорают по TTL, на вывод real есть лимиты; значения — параметры → [[Баланс]] |
| Купоны на подписки | **Нет**: купоны применяются только к арендам → [[Купоны]] |
