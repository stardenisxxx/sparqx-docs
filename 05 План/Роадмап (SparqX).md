# Роадмап

8 спринтов по 1 неделе. Команда — 3 разработчика. **MVP за 4 недели**, полный запуск — 8 недель.

| Спринт | Неделя | Что делается | Результат |
|---|---|---|---|
| 1 | 1 | Инфраструктура (6 серверов, K8s, PostgreSQL, Redis, RabbitMQ, MinIO, CI/CD) + 6 схем БД + Auth + базовый UI | Серверы подняты, БД созданы, авторизация работает |
| 2 | 2 | Device service (MQTT, парсинг `0x10`, токены, очереди) + IoT + дашборд устройств | Видим все устройства в реальном времени |
| 3 | 3 | User service + Rental service (lock, popup_sn, возврат) + UI пользователей | Аренда и возврат работают через API |
| 4 | 4 | Finance service (расчёт, идемпотентность) + сплит + payouts + финансовый UI | **MVP** — деньги считаются, сплит работает, пилот возможен |
| 5 | 5 | Coupon + Subscription service + UI купонов и подписок | Скидки применяются |
| 6 | 6 | Media service + коллекции + кампании + cron + UI рекламы | Реклама по расписанию на экранах |
| 7 | 7 | Support service + чат + вложения + notification + push + UI | Тикеты и уведомления работают |
| 8 | 8 | E2E + нагрузочное + мониторинг (Grafana, Alertmanager, ELK) + пилот | Система запущена, мониторинг работает |

> [!success] MVP после спринта 4
> Через 4 недели аренда, возврат, финансы и сплит работают — можно запускать пилот с реальными устройствами.
