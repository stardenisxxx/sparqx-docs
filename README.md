# SparqX — документация (Obsidian vault)

Git-версионируемый Obsidian-волт документации проекта SparqX. **Это единый источник правды** (заменяет копии в iCloud и `Claude/Projects`).

## Как работать
1. Откройте эту папку как vault в Obsidian (*Open folder as vault*).
2. Установите community-плагин **Obsidian Git** (Settings → Community plugins → Browse → "Obsidian Git").
3. Включите авто-commit/-pull/-push (рекомендуемые настройки — в шапке проекта).
4. Точка входа — [[🗺️ Карта проекта (SparqX)]].

## Структура
- `01 Бизнес-логика/` — бизнес-правила (баланс, бонусы, купоны, подписки, сплит, флоу аренды)
- `02 Сущности/` — схемы данных по доменам
- `03 Архитектура/` — система, микросервисы, БД, IoT, решения
- `04 Продукты/` — клиентские и админ-приложения
- `05 План/` — роадмап и открытые вопросы

## Совместная работа
Двое и более редакторов работают через Git: коммитьте часто, тяните перед началом, конфликты в `.md` разрешаются как обычный текст. Локальное состояние Obsidian (`workspace.json`, кэш) в Git не попадает — см. `.gitignore`.
