# Мобильная Admin

Урезанная версия [[Веб-Admin]] для работы на выезде — менеджеры и техники. React Native (Expo). **Без финансового модуля.**

## Bottom Tab Bar (5 вкладок)
| Вкладка | Содержимое |
|---|---|
| Dashboard | KPI: станции онлайн, активные аренды, выручка, алерты + mini area chart |
| Devices | Список станций, управление, диагностика павербанков |
| Alerts | Алерты в реальном времени + push при новом |
| Support | Тикеты — просмотр и ответ ([[Поддержка]]) |
| Profile | Настройки + выход |

Связано: [[Веб-Admin]], [[Дизайн-система]].
