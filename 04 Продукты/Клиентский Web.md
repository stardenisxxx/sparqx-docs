# Клиентский Web

Веб-версия [[Клиентское приложение|клиентского приложения]] для тех, кто без приложения. React + Vite. Responsive: 1440px Desktop + 390px Mobile. Полный функционал.

- Desktop: карта на 60% viewport + список ближайших станций справа (320px).
- Navbar: логотип + баланс + аватар (авторизованные) / Login + Sign Up (гости).
- Все экраны приложения адаптированы; на десктопе — модалки вместо отдельных экранов.
- WebSocket для чата в реальном времени.
