# Клиентское приложение SparqX

Приложение для клиентов: аренда павербанков, кошелёк, история, поддержка. React Native, тёмная и светлая темы.

## Bottom Tab Bar (5 вкладок)
| Вкладка | Ключевые экраны |
|---|---|
| Home (карта) | Карта + маркеры станций + карточка станции + скан QR + подтверждение аренды |
| Wallet | Баланс + бонусы + купоны + пополнение + транзакции + подписки |
| History | История аренд по годам; долги — красный фон |
| Support | Чат (In-app + WhatsApp + Telegram), статусы ✓✓ |
| Profile | Профиль + Payment methods + Settings + Information |

## Wallet — ключевой раздел
Карточка баланса (оранжевый градиент) с основным и бонусным балансом · active subscription с прогресс-баром и Renew · горизонтальный скролл купонов · последние 5 операций.

## Купоны и flows
QR-сканирование купона + ручной ввод, конфетти при добавлении ([[Купоны]]). Основные flows: аренда, возврат, купон QR, пополнение, подписка ([[Флоу аренды]]).
