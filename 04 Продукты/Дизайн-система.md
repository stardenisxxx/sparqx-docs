# Дизайн-система

Единый визуальный язык продуктов. Бренд — оранжевый `#FF5A00` на тёмном фоне.

## Палитра — Веб-Admin
| Token | HEX | Применение |
|---|---|---|
| brand/orange | #FF5A00 | Активное меню, CTA, акценты |
| brand/orange-muted | #8B3A00 | Неактивные пункты меню |
| bg/page | #121212 | Фон страницы |
| bg/sidebar | #2A2A2A | Фон бокового меню |
| bg/surface | #1E1E1E | Карточки, модалки |
| bg/elevated | #2C2C2C | Хедер таблиц, hover |
| text/primary | #FFFFFF | Основной текст |
| text/secondary | #BDBDBD | Вторичный текст |
| border/default | #333333 | Разделители |
| semantic/success | #4CAF50 | Онлайн, успех |
| semantic/warning | #FFC107 | Предупреждения |
| semantic/danger | #F44336 | Ошибки, офлайн |
| semantic/info | #1E88E5 | Информация, чат |
| chart/line | #FF5A00 | Линии графиков |
| chart/fill | #FF5A0033 | Заливка area chart |

## Боковое меню
Ширина 280px. Пункт — кнопка `border-radius:16px`, высота 60px. Активный: фон `#FF5A00` + белая точка справа. Неактивный: фон `#8B3A00`.

Применяется в [[Веб-Admin]], [[Мобильная Admin]], [[Клиентское приложение]], [[Клиентский Web]].
