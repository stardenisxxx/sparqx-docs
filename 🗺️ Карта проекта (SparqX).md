# 🗺️ SparqX — карта проекта

Платформа управления сетью станций **аренды павербанков** в крупном масштабе (10 000+ устройств, несколько стран, иерархия франшиз). Учёт станций и павербанков в реальном времени, аренды, финансовый сплит по уровням, купоны, подписки, реклама на экранах и поддержка.

> [!info] Четыре продукта на одном бэкенде
> Веб-Admin и Клиентское приложение — это MVP (P0). Мобильная Admin и Клиентский Web — следующий приоритет (P1). Всё работает поверх 10 микросервисов и PostgreSQL.

## Быстрые факты
- Масштаб: **10 000+ устройств**, несколько стран, иерархия франшиз.
- Аренда в одном месте — **возврат в любую станцию** (ключевое решение БД).
- Сплит прибыли по цепочке Суперадмин → Дистрибьютер → Франшиза, процент фиксируется **исторически**.
- БД: PostgreSQL 16, **6 схем**, мягкие межбазовые связи (целостность проверяет бэкенд).
- IoT: Alibaba Cloud MQTT, телеметрия `0x10` каждые 20 минут, токены кэшируются в Redis на 7 дней.
- 6 типов купонов, QR-сканирование, сегментация по странам/городам/партнёрам.
- План: **8 спринтов**, команда 3 разработчика, MVP за 4 недели.
