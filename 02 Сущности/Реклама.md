# Реклама

Медиафайлы и кампании, которые показываются на экранах станций. Схема `media`, файлы в MinIO.

## Таблицы (схема `media`)
| Таблица | Ключевые поля | Назначение |
|---|---|---|
| media_assets | id PK, name, type (video/image/banner/h5), file_url, file_size | Медиафайлы в MinIO |
| campaigns | id PK, name, status, starts_at, ends_at | Рекламные кампании |
| campaign_assets | id PK, campaign_id FK, asset_id FK, display_order, display_seconds | Плейлист кампании |
| campaign_targets | id PK, campaign_id FK, target_type, country_code, franchise_id, station_imei | Таргетинг |
| campaign_schedules | id PK, campaign_id FK, day_of_week, time_from, time_until, timezone | Расписание показа |
| device_content_log | id PK, campaign_id FK, station_imei, status, sent_at, confirmed_at | Лог доставки на устройства |

## Логика
- **Сеты (плейлисты)** медиафайлов с порядком и длительностью показа каждого.
- **Таргетинг** одновременно: страна + партнёр/франшиза + конкретные станции.
- **Расписание**: сразу или по дате, часы, дни недели, timezone, повтор.
- Доставка на станции через media-service; статус и retry failed видны в [[Веб-Admin]].

Связано: [[Микросервисы]], [[Веб-Admin]].
