# Финансы

Деньги системы: страны и валюты, тарифы, платежи, сплит, выплаты, подписки, баланс клиента. Схема `finance`.

## Таблицы (схема `finance`)
| Таблица | Ключевые поля | Назначение |
|---|---|---|
| countries | code PK, name, currency_code, currency_symbol, timezone | Справочник стран и валют |
| tariffs | id PK, country_code FK, price_per_hour, min_charge | Один [[Подписки и тарифы\|тариф]] на страну |
| payments | id PK, order_id FK, customer_id FK, tariff_id FK, amount, status | Платёж за аренду |
| revenue_splits | id PK, payment_id FK, franchise_id FK, percent, amount | [[Франшизы и сплит\|Сплит]] (percent — историч. снимок) |
| payouts | id PK, franchise_id FK, amount, currency_code, status | Выплаты партнёрам |
| subscription_plans | id PK, country_code FK, name, duration_days, price, discount_pct | Планы [[Подписки и тарифы\|подписок]] |
| customer_subscriptions | id PK, customer_id FK, plan_id FK, expires_at, status | Активные подписки |
| balance_transactions | id PK, customer_id FK, pocket (real\|bonus), amount (знаковая), type, status, payment_id FK, idempotency_key uniq | Журнал движений [[Баланс\|баланса]] — источник правды |

> [!warning] Идемпотентность платежей
> Finance-service проверяет `GET processing:{order_id}` в Redis — повторная обработка события из очереди не создаёт два платежа. См. [[Ключевые решения]].
