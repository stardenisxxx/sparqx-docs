# Клиенты и франшизы

Пользователи системы: клиенты приложения, администраторы и иерархия партнёров. Схема `users`.

## Таблицы (схема `users`)
| Таблица | Ключевые поля | Назначение |
|---|---|---|
| admin_users | id PK, email, password_hash, role, franchise_id FK | Администраторы системы |
| franchises | id PK, name, parent_id FK (self), revenue_pct, type | Иерархия Суперадмин → Дистрибьютер → Франшиза |
| customers | id PK, phone unique, name, balance, is_blocked | Клиенты приложения |
| sessions | id PK, user_id FK, user_type, token, expires_at | Активные сессии (admin и customer) |
| roles_permissions | role, permission | Матрица прав доступа |

## Иерархия (self-FK)
`franchises.parent_id` ссылается на саму таблицу — отсюда дерево партнёров и расчёт [[Франшизы и сплит|сплита]]. `revenue_pct` — базовая ставка уровня (в сплит фиксируется исторический снимок).

## Клиенты
Регистрация по телефону, баланс + бонусный баланс из [[Купоны|купонов]], возможность блокировки/заморозки из [[Веб-Admin]].
